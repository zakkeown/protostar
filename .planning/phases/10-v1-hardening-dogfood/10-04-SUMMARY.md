## Deviations from Plan

- `packages/artifacts/src/index.ts` currently exports TypeScript interfaces, not named Zod schemas. The appendix generator therefore defines local Zod schema equivalents matching those interfaces and converts them with `zod-to-json-schema`.
- The plan suggested `tsx`; this repo did not have `tsx`, so the script runs under Node 22 type stripping with `node scripts/gen-schema-appendix.ts`.
- The CLI writes help to stderr in the current build. Snapshots and the drift contract preserve that behavior instead of rewriting the CLI surface in this docs slice.
